# 📊 Análise de Vendas com SQL e Power BI

Projeto completo de análise de dados, desde a construção do banco em SQL até a criação de um dashboard interativo no Power BI.

## 🛠️ Tecnologias utilizadas
- SQL (SQLite)
- Power BI

## 📈 Análises realizadas
- Ranking de clientes por volume de compras
- Ranking de clientes por faturamento
- Ranking de produtos por quantidade vendida
- Faturamento por produto
- Distribuição de vendas por cidade
- Tendência de faturamento ao longo do tempo

## 📁 Estrutura do projeto
- 01_schema.sql → criação das tabelas
- 02_inserts.sql → inserção de dados
- banco.db → banco de dados
- dashboardvendas.pbix → dashboard Power BI

## 💡 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar análise de dados, modelagem e visualização, conectando SQL ao Power BI para gerar insights de negócio.
